Technology Used

Websites promote you 24/7: No employee will do that.

HTML

HTML provides the base structure for a web page. HTML code ensures that all the content on a website is properly formatted. Without HTML, a browser couldnt display text or load images and other elements.

CSS

CSS define the style and aesthetics of a web page. It specifies the appearance of that structure. This includes page layouts, colours, fonts and element positioning. It makes your website look good.

Programming languages

JavaScript, PHP, Phython, Java are the programming langauges are used in website development. They provide a set of grammatical rules to instruct a computer to perform specific functions.

Web Development Frameworks

Angular and Node.js are the frameworks used in website development. There are both front-end and back-end frameworks that streamline the development process

Libraries

React.js, Vue.js and FacebookSDK are the libraries used in web development process. Libraries are collections of files, programs, routines, scripts or functions that can be integrated when writing code

Databases

SQL and NoSQL are databases used in define and store data . It provide a consistent, organized structure for storing and retrieving large amounts of data.
